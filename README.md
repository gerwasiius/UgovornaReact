# AutoDoc

Ovaj projekat sadrži Blazor frontend za upravljanje dokumentima. Sve modalne komponente nasljeđuju `ModalBase` klasu koja centralizuje logiku otvaranja i zatvaranja modala. Poslovna logika je smještena u `.razor.cs` fajlove radi bolje čitljivosti i održavanja. Stranica `Placeholders` takođe koristi code-behind pristup.
